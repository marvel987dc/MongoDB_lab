COMP229 - Lab 2: MongoDB and JavaScript Integration

Objective:

In this lab, you will integrate MongoDB with Node.js to answer a series of questions by querying data from MongoDB and processing the results in JavaScript.

Your tasks involve writing MongoDB queries and applying JavaScript logic to return answers for each of the questions. Two collections are provided in the MongoDB database: countries and medals. You will use these collections to answer the following questions.

Required Packages:

  • mongodb
  • dotenv

Instructions:

1. Environment Setup:

  • Set up your project and install the necessary packages (mongodb and dotenv) using npm.
  • Use a .env file to store your MongoDB connection string.

2. Understanding the Structure:

  • The MongoDB database contains two collections: countries (for population and geographical data) and medals (for Olympic medal data). The questions will involve querying these collections.
  • Each question has a MongoDB query part (to fetch the data) and a JavaScript processing part (to compute the answer based on the data fetched).

3. Tasks:

Question 1: What is the Capital of the country that has the largest 2022 population?

  • MongoDB Part: Fetch all countries.
  • JavaScript Part: Identify the country with the largest 2022 population and return its capital.

Question 2: What is the total population of countries in Europe as of 2022?

  • MongoDB Part: Query for all European countries.
  • JavaScript Part: Sum their populations.

Question 3: How many medals did Canada win in the 2024 Olympics?

  • MongoDB Part: Fetch data about Canada from the medals collection.
  • JavaScript Part: Return the number of medals won.

Question 4: How many gold medals were won by countries in Asia?

  • MongoDB Part: Query for countries from Asia in the medals collection.
  • JavaScript Part: Count and return the total number of gold medals won.

Question 5: Which country with a population under 5 million won the most medals in the 2024 Olympics?

  • MongoDB Part: Query for countries with a population under 5 million in 2022.
  • JavaScript Part: Determine which of these countries won the most medals.

Question 6: What is the total number of bronze medals won by countries in Europe?

  • MongoDB Part: Query for European countries in the medals collection.
  • JavaScript Part: Sum the number of bronze medals won.

Question 7: What is the total population of countries in Africa that participated in the 2024 Olympics?

  • MongoDB Part: Query for African countries that participated in the 2024 Olympics.
  • JavaScript Part: Sum their populations.

Question 8: Which country with more than 50 million people in 2022 won the fewest medals in the 2024 Olympics?

  • MongoDB Part: Query for countries with a population greater than 50 million.
  • JavaScript Part: Find which of these countries won the fewest medals.

Question 9: Which country with a population under 1 million has the highest medal count in the 2024 Olympics?

  • MongoDB Part: Query for countries with a population under 1 million.
  • JavaScript Part: Determine which of these countries has the highest medal count.

Question 10: Which country had the highest medal-to-population ratio in the 2024 Olympics?

  • MongoDB Part: Query for all countries that participated in the 2024 Olympics.
  • JavaScript Part: Calculate and return the country with the highest ratio of medals to population.

Final Steps:

  • Complete the code for each question by splitting it into a MongoDB part and a JavaScript part.
  • Test your functions to ensure they return the correct answers for each question.
  • If you finish early, explore how you could optimize your MongoDB queries.

Good luck, and remember to break down each task into manageable parts!
